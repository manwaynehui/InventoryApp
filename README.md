# Inventory Management App

A Flask-based inventory system with a customer-facing shop and an admin dashboard for order management.

# How to start

Follow these steps to get the project running on your local machine:

# 1. Install Dependencies
Make sure you have Python installed, then run in terminal:
- pip install requirements.txt


# 2. Run the setup_db.py in terminal or IDE
It will create an instance folder with the database:
- python setup_db.py
  
# 3. Run the app.py in terminal or IDE:
- python app.py

# 4. Open the link of local host and it will redirect you to the site.

For logging into admin, the password is: admin123
